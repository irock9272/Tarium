# Profiles Directory

This folder stores per-profile browser data created at runtime.

## What is isolated per profile

- Cookies and site storage
- History
- Bookmarks
- Plugin enabled/disabled state

This lets you keep separate browsing contexts (e.g., personal vs work).

## Typical usage

1. Launch Tarium.
2. In **Profile Manager**, create/select a profile.
3. Browse normally — data remains scoped to that profile.

## Notes

- Deleting a profile from Profile Manager removes its local data.
- You generally should not edit profile internals manually while Tarium is running.
