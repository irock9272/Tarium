# Icons Directory

This folder contains toolbar/app image assets used by Tarium.

## Current assets

Examples include:
- navigation (`back.png`, `forward.png`, `refresh.png`, `home.png`)
- actions (`new_tab.png`, `bookmark.png`, `history.png`, `plugin.png`, `go.png`)
- app/menu (`logo.png`, `hambuger_dropdown.png`)

## Notes

- Keep filenames stable unless you also update references in the Python UI code.
- Prefer consistent style and contrast so icons remain visible in dark themes.
